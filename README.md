It allows:

- Adding, editing, and deleting **classes**

There are **4 core components** in the app:

1. Add Class  
2. View Classes  
3. Add Department  
4. View Departments

- `Sifarnici` (Codebooks)
- `Stavke sifarnika` (Codebook items)
- `Razredi` (Classes)
- `Odeljenja` (Departments)
